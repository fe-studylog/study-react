Regular function

            hello = function() {
                return "Hello World!";
            }

Arrow function

            hello = () => {
                return "Hello World!";
            }

Implicit Return

            hello = () => return "Hello World!";

Arrow Function with Parameters

            hello = (val) => "Hello" + val;

Arrow Function Without Parentheses

If you have only one parameter, you can skip the parentheses as well. 

            hello = val1 => "Hello" + val1;
